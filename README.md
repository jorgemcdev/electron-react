# React / Electron starter
Electron application boilerplate based on React, Webpack and package to OS-specific

## Development Mode

Terminal 1
```
$ npm run watch
```

Terminal 2

```
$ npm start
```

## Build
```
$ npm run build
```

## Package your Electron app into OS-specific

Mac
```
$ npm run package-mac
```
Windows
```
$ npm run package-win
```
```
$ npm run package-linux
```
